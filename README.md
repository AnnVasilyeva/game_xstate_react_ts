# Game - React + TypeScript + xState + SCSS

Игра на упорядочивание цифр.

### Предварительные требования

- Node.js 18+
- npm, yarn, or pnpm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will open at `http://localhost:5173`

## Механики

Цифры можно переставлять двумя способами:

1. Drag and drop

2. Click swap

## Проверка ответа

На экране должна быть кнопка Проверить. После нажатия:

- если порядок совпадает с целевой последовательностью, показать сообщение: “Победа!”;
- если порядок неверный, показать сообщение: “Порядок не соответствует заданному, попробуй еще раз!”;
- после “Победы” перестановки должны быть заблокированы;
- должна быть кнопка “Новая игра”.
